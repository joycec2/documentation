utils module

Provides utility functions to support text parsing, including logic to extract numbered questions from LLM responses.

Functions:

  • extract_numbered_questions: Extracts ordered question strings from a given text.

llm.agents.utils.extract_numbered_questions(content: str, marker: str = '</think>') → List[str][source]

Extracts numbered questions from the given content after the specified marker.

Useful for parsing multi-step LLM responses that contain ordered steps or subqueries.

Parameters:

  • content (str) -- The complete string to extract questions from.

  • marker (str, optional) -- Substring indicating where to begin extraction. Defaults to "</think>".

Returns:

A list of cleaned, ordered question strings.

Return type:

List[str]
